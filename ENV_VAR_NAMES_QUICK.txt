================================================================================
HOW TO FILL:
================================================================================

1. Click in the "Name" field (red border)
2. Type the name (e.g., NODE_ENV)
3. Verify the "Value" field is correct
4. Click "DONE" button
5. Repeat for each row

================================================================================
IMPORTANT:
================================================================================

✓ Name = Variable name only (no spaces, no = sign)
✓ Value = The actual value (already shown in grey boxes)
✓ No quotes needed
✓ Case sensitive - use exact capitalization

================================================================================

Row 1:
  Name:  NODE_ENV
  Value: production
  → Click "DONE"

Row 2:
