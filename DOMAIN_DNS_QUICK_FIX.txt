================================================================================
  QUICK FIX: Invalid Domain Configuration
================================================================================

PROBLEM:
  Domain shows "Invalid Configuration" in Vercel

CAUSE:
  DNS records not pointing to Vercel

================================================================================
QUICK STEPS:
================================================================================

1. GET DNS RECORDS FROM VERCEL
   → Go to: Settings → Domains
   → Click on suafast.live
   → Click "Learn more" or "Edit"
   → Copy the DNS records shown

2. UPDATE DNS AT YOUR REGISTRAR
   → Log into where you bought suafast.live
   → Find "DNS Settings" or "DNS Management"
   → Add these records:

   For suafast.live (root):
     Type: A
     Name: @ (or leave blank)
     Value: [IP from Vercel - usually 76.76.21.21]
   
   For www.suafast.live:
     Type: CNAME
     Name: www
     Value: cname.vercel-dns.com

3. SAVE AND WAIT
   → Save DNS changes
   → Wait 15-60 minutes
   → Click "Refresh" in Vercel

4. VERIFY
   → Status should change to "Valid Configuration"
   → Domain should work!

================================================================================
COMMON REGISTRARS:
================================================================================

GoDaddy:
  → My Products → DNS → Manage DNS

Namecheap:
  → Domain List → Manage → Advanced DNS

Cloudflare:
  → DNS → Records → Add record

Google Domains:
  → DNS → Custom records

================================================================================
TROUBLESHOOTING:
================================================================================

Still invalid after 1 hour?
  → Check: https://dnschecker.org
  → Search: suafast.live
  → Verify records match Vercel's values

Wrong records?
  → Delete old A/CNAME records
  → Add new ones from Vercel

Need help?
  → Check Vercel domain settings for specific errors
  → Contact your domain registrar

================================================================================
AFTER DNS IS FIXED:
================================================================================

1. Update NEXTAUTH_URL in Vercel:
   → Settings → Environment Variables
   → Change to: https://suafast.live
   → Redeploy

2. Test:
   → Visit: https://suafast.live
   → Should work!
